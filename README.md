# Traffic Accident Data Analysis

This project aims to analyze traffic accident data to uncover patterns, trends, and insights that can improve road safety and inform policy makers. By utilizing various data visualization techniques, we will explore factors contributing to accidents and identify hotspots for the most severe incidents.

## Tools Used
- **Python**: Core programming language for analysis.
- **Pandas**: For data manipulation and analysis.
- **Folium**: For creating interactive maps.
- **Matplotlib** and **Seaborn**: For data visualization.
- **Geopandas**: To handle geospatial data.

## Expected Insights
- Identification of high-risk areas for traffic accidents.
- Analysis of the impact of weather conditions on accident frequency.
- Trends in accident rates over time and by different demographics.
